
# Detección de Fraudes en Seguros de Autóviles
Proyecto de la fase dos de BEDU python - Data Science

## Objetivo del Proyecto
- Investigar, analizar y evaluar el impacto que tienen los fraudes en los seguros de automóviles en la sociedad.
- Conocer el perfil de las personas que son más propensas a cometer fraude 
- Buscar datasets con informacion util para analizar el problema de los fraudes en los seguros. 
- En esta primera fase, se busca realizar un análisis previo y una limpieza de los datos.
- En sesiones futuras, se buscará aplicar diferentes modelos de aprendizaje máquina para poder predecir fraudes en automóviles para poder prevenirlos y lograr una reducción en los costos de operación de las aseguradoras

## Librerias utilizadas
 - Pandas
 - Numpy
 - Seaborn
